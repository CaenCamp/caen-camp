Micro Frontends et Web Components

Jean-Baptiste Oellers
  • javascript

Votre projet front est devenu trop gros et trop complexe : il est temps de le découper. Mais comment faire ?

Les Micro Frontends peuvent apporter des éléments de réponse. Autour d'un exemple simple, nous ferons évoluer un front monolithique vers une architecture Micro Frontends à l'aide d'une technologie standard du web : les Web Components.
